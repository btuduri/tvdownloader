# Dépendances de TVDownloader #

## Version 0.6 ##

  * Python (version 2)
  * PyQt (>= 4.7)
  * rtmpdump
  * msdl
  * python-crypto ou pycrypto
  * python-mechanize

## Version 0.5 ##

  * Python (version 2)
  * PyQt (>= 4.7)
  * rtmpdump
  * msdl
  * python-crypto ou pycrypto

## Version < 0.5 ##

  * Python (version 2)
  * PyQt (>= 4.7)
  * rtmpdump
  * msdl (inclus dans le paquet)
